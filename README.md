# Netflix Clone 🎬

A pixel-perfect clone of the Netflix India landing page, 
built using only HTML5 and CSS3.

## Live Demo
https://khushiish11.github.io/netflix-clone

## Features
- Responsive design for all screen sizes
- CSS radial gradient curve section transition
- Trending movies grid with numbered overlays
- Hover effects on buttons and input fields
- FAQ section layout
- Multi-language footer selector
- Mobile-first media queries

## Technologies Used
- HTML5 (semantic elements)
- CSS3 (Flexbox, Grid, Gradients, Animations)
- Font Awesome icons

## What I learned
- Complex CSS positioning and z-index management
- CSS radial gradients for section transitions
- Responsive grid layouts using auto-fit/minmax
- Fixed a tricky positioning bug with trending 
  numbers using position: relative on parent elements
- Implemented media queries for tablet and mobile views
